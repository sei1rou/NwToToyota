NwToToyota.exeは、健診システムからトヨタ販売健保用のデータを作成するプログラムです。



【作成手順】
１．データ抽出「A85 トヨタ販売健保定健データ提出用」パターンで、対象データを出力
　　出力データの形式はタブ区切り、""はなし

２．出力したファイルをNwToToyota.exeにドロップ
　　出力したファイルと同じディレクトリに
　　log.txtが作成され
　　トヨタ東京販売ホールディングス[日付]のフォルダに
　　会社毎の健診データファイルと受診者名簿が作成されます。
　　

※変換エラーはlog.txtに記載されているので、必ず確認してください。
